 Orthogroup Analysis1. Installation* Orthofinder has been Installed using conda install orthofinder.2. Data Download * "protein_primaryTranscriptOnly.fa.gz" files were downloaded in a new directory for Arabidopsis thaliana, Oryza sativa (rice), and Amborella trichopoda using : https://data.jgi.doe.gov/refine-download/phytozome?genome_id=291,323,447* Files were decompressed and renamed as  Athaliana.fa, Osativa.fa, Atrichopoda.fa.3. Run Orthofinder* orthofinder ran using orthofinder –f  name of the directory containing my fasta files.* I got result file in the directory where I had my fasta files and got my tsv files in Phylogenetic_Hierarchical_Orthogroups folder.
